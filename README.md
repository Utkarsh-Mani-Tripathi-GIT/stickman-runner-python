# Stickman Runner

**Author:** Utkarsh Mani Tripathi
**LinkedIn:** [linkedin.com/in/utkarsh-mani-tripathi](https://www.linkedin.com/in/utkarsh-mani-tripathi/)

A fast-paced infinite runner built with Python and Pygame. This project demonstrates procedural generation for game assets, state management, and real-time audio synthesis.

## Features

- Procedural Character Animation: Animated run, jump, duck, and death sprites drawn in real-time.
- Escalating Threat Systems: Enemy jets and homing missiles programmed with specific flight paradigms.
- Particle Engine: Mathematically simulated explosions, sparks, and exhaust particles.
- 8-Bit Synthesizer: Custom audio synthesis for jump, land, coin, and damage waveforms.

### Prerequisites

- Python 3.8+: https://www.python.org/downloads/
- Pygame Library: https://www.pygame.org/

## Installation and Execution

1. Clone or clone the repository to your local drive.
2. Initialize and activate a Python virtual environment.
3. Secure the dependencies by running `pip install -r requirements.txt`.
4. Run the mainline program by calling `python stickman_runner.py`.

## Technical Architecture

- Language: Python
- Graphic Library: Pygame
- Design Pattern: Object-Oriented Component Architecture.
- Render Paradigm: Pure geometry draw calls; no external image assets.
